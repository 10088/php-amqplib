AMQPDecimal¶

AMQP protocol decimal value.

Values are represented as (n,e) pairs. The actual value is n * 10^(-e). From 0.8 spec: Decimal values are not intended to support floating point values, but rather business values such as currency rates and amounts. The 'decimals' octet is not signed.
